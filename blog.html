1.Discuss the scope of var, let & const?

var:

Var is a function-scoped, not block-scoped.If declare a variable with var inside a function, it is only accessible within that function, and not outside of it.

Let:

let is a block scope.But inside the block element the value of let can be changed & it will be accessible only within that block.

Const:

Const is also a block scope.Therefore, they are only accessible within the block of code where they are defined.

2.Tell us the use case of null & undefined?

Null:

"Null" is a value that represents the intentional absence of any value or object.

Use case of Null:

  • 1.It's often used to indicate that a variable or property deliberately has no value.

Undefined:

A variable that has been declared but hasn't been assigned a value is considered'undefined'.

Use case of Undefined:

  • 1. Property that doesn't exists on an object will give you Undefined.
  • 2. Accessing array elements outside of the index range.
  • 2. If return has nothing on the right side will return Undefined.

3.What do you mean by rest API?

API stands for Application Programming Interface.The common API use by most mobile and web application is talk to server which is called REST. REST stands for Representational State Transfer.It's not a specification but it is a set of rule.It has been the common standard for building web API. An API that follows the REST is called RESTfull API. The API get all products should be slash Products.

HTTP Methods

The four most common HTTP methods used in REST are:

  • GET: Retrieve data from the server
  • POST:Create a new resource on the server.
  • PUT: Update an existing resource on the server.
  • DELETE: Remove a resource from the server.
